# AegisAI – Autonomous Defence and Self-Healing System

## Overview

AegisAI is a real-time intelligent monitoring and automation system developed to monitor system performance, detect abnormal behaviour, and improve system stability using machine learning concepts and automated response handling.

The system continuously tracks CPU usage, RAM usage, Disk activity, Network usage, and running processes in real time through an interactive dashboard interface. It also performs anomaly detection using machine learning algorithms to identify unusual system behaviour and trigger intelligent monitoring responses.

This project was developed as a final-year BCA project focused on practical system monitoring, automation workflows, real-time visualization, and intelligent response handling.

4. The frontend dashboard visualizes real-time monitoring data.

# Machine Learning Integration

The project uses the Isolation Forest algorithm from scikit-learn for anomaly detection. The model helps identify unusual system behaviour patterns based on real-time monitoring data collected from the system.
